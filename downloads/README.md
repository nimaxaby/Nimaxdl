# فایل‌های دانلود شده

## نحوه دانلود قطعات
1. وارد پوشه هر فایل شوید
2. فایل `links.csv` را باز کنید
3. روی لینک هر قطعه کلیک کنید

## نحوه چسباندن قطعات
فایل‌های `.z01`, `.z02`, `.zip` توسط نرم‌افزارهای معمولی قابل باز شدن هستند.

### در ویندوز:
- روی فایل `.zip` کلیک راست کنید
- گزینه **Extract Here** را انتخاب کنید

### در اندروید:
- با **ZArchiver** یا **RAR** مستقیماً فایل `.zip` را باز کنید

### در لینوکس/مک:
```bash
unzip file.zip
```

## نکته
اگر فایل با رمز محافظت شده، هنگام باز کردن رمز را وارد کنید.
